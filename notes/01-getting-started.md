# Getting started

## Installation
```bash
$ curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```

## Troubleshooting
```bash
rustc --version
```

Checks the current version installed in the system.

## Updating and Uninstalling
```bash
rustup update
```
Updates the rust version to the current stable version.

```bash
rustup self uninstall
```
Uninstalls rust and rustup.

## Local Documentation
```bash
rustup doc
```
